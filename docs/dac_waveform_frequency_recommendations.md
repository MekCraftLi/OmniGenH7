# DAC 波形最高建议频率记录

日期：2026-05-17

## 评估前提

- MCU：STM32H723ZG 内部 DAC，主输出为 `PA4 / DAC1_OUT1`。
- DAC 更新采样率按当前目标上限 `1 MS/s` 评估。
- 默认存在外部模拟重建低通滤波器和高速运放缓冲/放大级。
- 下表的“最高建议频率”指比较理想的工程建议上限，不是理论 Nyquist 极限。
- 对 AM/FM，“频率”主要指载波频率；FM 还必须同时限制最高瞬时频率。
- 对 Sinc/扫频，“频率”分别指 Sinc 带宽/主瓣参数和扫频终止频率。

## 最高建议频率

| 波形 | 最高建议频率 | 扩展可用上限 | 100 kHz 评价 | 备注 |
|---|---:|---:|---|---|
| Sine 正弦 | `50 kHz` | `100 kHz` | 可用，但需要重建滤波和幅度校准 | `100 kHz` 时只有 `10` 点/周期，适合窄带正弦，不适合作为通用高质量指标。 |
| AM 调幅 | `30 kHz` 载波 | `50 kHz` 载波 | 不建议作为高质量输出 | 调制带宽应受控，建议最高边带保持在约 `50 kHz` 内。 |
| FM 调频 | `20~30 kHz` 载波 | `50 kHz` 载波，频偏要小 | 不建议作为高质量输出 | 建议最高瞬时频率 `f_carrier + f_deviation <= 50 kHz`。 |
| Triangle 三角波 | `10~20 kHz` | `50 kHz` | 不建议，形状明显圆滑/失真 | 三角波依赖高次谐波，频率升高后外部滤波会削弱边角。 |
| Square 方波 | `10 kHz` | `20~50 kHz`，边沿会被圆滑 | 不建议作为理想方波 | 方波谐波需求最高，`100 kHz` 的 5 次谐波已到 Nyquist。 |
| Sawtooth 锯齿波 | `10 kHz` | `20 kHz` 左右 | 不建议 | 锯齿波包含偶次和奇次谐波，对采样率和滤波器要求高。 |
| Sinc | `100 kHz` 带宽/主瓣参数 | 视输出幅度和滤波器而定 | 当前仿真中表现较好 | Sinc 不是周期波形，关注的是频谱带宽而不是周期频率。 |
| Sweep 扫频 | 终止频率 `50 kHz` | 终止频率 `100 kHz` | 可展示，但高频端质量下降 | 若要全程质量接近一致，建议扫频终点限制在 `50 kHz`。 |
| Arbitrary 任意波 | 最高有效频谱 `50 kHz` | 最高有效频谱 `100 kHz`，质量下降 | 取决于波形频谱 | 快速边沿、尖峰、窄脉冲类任意波应按方波/锯齿波保守处理。 |

## 采样点约束

在 `1 MS/s` DAC 更新率下：

```text
100 kHz = 10 points/cycle
50 kHz  = 20 points/cycle
20 kHz  = 50 points/cycle
10 kHz  = 100 points/cycle
```

工程判断：

- 正弦、窄带 AM/FM 可以在较少点数下通过模拟滤波改善波形。
- 三角波、方波、锯齿波需要更多谐波，建议频率明显低于正弦类波形。
- `100 kHz` 更适合作为正弦/窄带测试上限，不适合作为所有波形的高质量规格。

## 模拟滤波器建议

总原则：

- `1 MS/s` DAC 的主要镜像分量会出现在 `1 MHz - f_signal`、`1 MHz + f_signal` 附近，重建滤波器需要压制这些镜像。
- 正弦、AM、FM、扫频适合低通重建滤波器。
- 三角波、方波、锯齿波需要保留谐波；低通太陡会让波形变圆，太宽又会留下 DAC 阶梯和镜像。
- 单纯 RC 无源滤波器适合低成本/低频预滤波；高质量输出建议使用有源滤波器或“无源 RC + 高速运放缓冲”。
- 对 `50 kHz` 以上输出，建议滤波后必须有高速运放缓冲，避免负载改变滤波器响应。

| 波形 | 建议滤波器 | 推荐阶数 | 有源/无源 | 建议截止频率 | 说明 |
|---|---|---:|---|---|---|
| Sine 正弦 | 低通重建滤波器，Butterworth 或 Bessel | 2~4 阶 | 优先有源；低成本可用无源 RC + 缓冲 | `2~3 * f_signal`，高频正弦可取 `150~250 kHz` | Butterworth 幅频平坦；Bessel 相位更好但滚降较慢。 |
| AM 调幅 | 低通重建滤波器，Butterworth 或 Bessel | 2~4 阶 | 有源更合适 | `1.5~3 * (f_carrier + f_mod)` | 截止频率必须覆盖最高边带，否则包络会被压缩。 |
| FM 调频 | 低通重建滤波器，Bessel 或 Butterworth | 2~4 阶 | 有源更合适 | `1.5~2.5 * (f_carrier + f_deviation + f_mod)` | FM 对相位/群时延更敏感，优先 Bessel；频偏大时必须降低载波。 |
| Triangle 三角波 | 宽带低通或椭圆/Butterworth 折中滤波 | 2~3 阶 | 有源或无源 LC/RC + 缓冲 | 至少覆盖 `5~7` 次谐波，约 `5~7 * f_signal`，但不超过约 `450 kHz` | 截止太低会变成近似正弦，截止太高会保留阶梯噪声。 |
| Square 方波 | 不建议用重建低通追求理想方波；可用宽带缓冲 + 轻度低通 | 1~2 阶 | 高速比较器/整形更合适；模拟滤波只做限带 | 若要保边沿，`> 5~7 * f_signal`；若要限带，按目标上升时间选 | 高质量方波应使用 GPIO/定时器/比较器输出，不应依赖 DAC + 低通。 |
| Sawtooth 锯齿波 | 宽带低通，尽量线性相位 | 2~3 阶 | 有源更合适 | 至少覆盖 `5~10 * f_signal`，受 `450 kHz` 上限限制 | 锯齿波谐波多，滤波后高频时会明显圆滑。 |
| Sinc | 低通重建滤波器，Butterworth 或 Bessel | 2~4 阶 | 有源更合适 | `1.2~2 *` 目标带宽 | Sinc 关注频谱带宽，截止频率应按带宽而不是周期频率设。 |
| Sweep 扫频 | 固定低通重建滤波器，Butterworth 或 Bessel | 2~4 阶 | 有源更合适 | `1.5~2 * f_stop` | 如果扫到 `100 kHz`，截止可取 `150~250 kHz`；高频端幅度需要校准。 |
| Arbitrary 任意波 | 可切换滤波器组 | 2~4 阶 | 有源滤波器组 + 模拟开关 + 输出缓冲 | 按最高有效频谱选择档位 | 建议提供多个滤波档位，而不是一个滤波器覆盖所有波形。 |

## 建议滤波器档位

结合当前项目已有 `FILTER_SEL0`、`FILTER_SEL1`、`FILTER_EN` 和 `OUTPUT_MUTE` 引脚规划，建议做可切换滤波器组：

| 档位 | 建议截止频率 | 推荐类型 | 适用波形/频段 |
|---|---:|---|---|
| Bypass / Wide | `> 500 kHz` 或直接旁路 | 输出缓冲为主 | 方波、低保真任意波、调试 DAC 阶梯输出。 |
| Low | `20~30 kHz` | 2~4 阶有源低通 | `1~10 kHz` 正弦、AM/FM、三角波。 |
| Mid | `100~150 kHz` | 2~4 阶有源低通 | `10~50 kHz` 正弦、AM/FM、扫频。 |
| High | `200~250 kHz` | 2~4 阶有源低通 | `50~100 kHz` 正弦、窄带 AM、扫频高频端。 |

实现建议：

- 切换滤波器前先拉起 `OUTPUT_MUTE`，稳定后再打开输出，避免切换噪声。
- 每个滤波档位需要独立幅度校准；高频端尤其需要补偿 DAC 零阶保持和滤波器衰减。
- 若只做一个滤波器，优先选 `150~200 kHz` 的 3~4 阶有源低通，用于正弦/AM/FM/扫频；方波和锯齿波质量不作为主要指标。
